Projects

All of these projects are a work in progress. Things will change, and more projects will be added as time goes on.

Simple Password Manager

I created a simple password manager using Python, and some simple modules like cryptography, and CustomTkinter. It tracks your username, password, and the app/site it belongs to. Then it stores all of that info into an encrypted .json file that is created after running the app.

Password Manager - Android Port

APK version of the desktop password manager I created. Same principal applies, everything gets encrypted, and stored locally. For the port I utilized Kivy and buildozer in order to get things running.

Tailwind Demo

Testing grounds for tailwind/node.js functionality - DB hosted w/ MongoDB Atlas Mainly used for my own personal testing of tailwind, and node.js functionality. The site has a working blog functionality that creates posts and stores them in the database.
